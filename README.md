# College Ads Sorter

Chrome extension that classifies unread Gmail college marketing emails and moves likely ads into a `College Ads` label.

## Install Locally

1. Download or clone this repo.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer mode**.
4. Click **Load unpacked**.
5. Select this folder.

The extension uses Chrome's built-in local AI support, so Chrome may require enabling the Prompt API / Gemini Nano flag before classification works.
