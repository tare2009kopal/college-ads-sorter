MailSift
Privacy Terms

A calmer Gmail inbox for college application season.

MailSift is a Chrome extension that helps students filter college admissions promotional emails into a Gmail label called College Ads while keeping important school, testing, scholarship, counseling, research, and lab messages protected.

View on Chrome Web Store

What MailSift Does

  • Connects to Gmail with the user's permission.
  • Scans inbox messages for college admissions campaign signals.
  • Moves matched promotional admissions emails into College Ads.
  • Protects user-selected categories, colleges, senders, and research/lab topics.
  • Shows a report of what was moved, protected, checked, or skipped.

How It Runs

MailSift runs as a Chrome extension. It does not use a separate cloud backend. Gmail access is used only to classify and organize messages according to the user's settings.

MailSift is an independent student project and is not affiliated with Gmail, Google, or any college or university.
