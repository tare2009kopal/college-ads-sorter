Privacy Policy

Effective date: July 3, 2026

Single Purpose

MailSift helps users filter college admissions promotional emails from Gmail while protecting important school, testing, scholarship, counseling, research, and lab-related messages.

Data MailSift Accesses

MailSift may access Gmail message metadata and content, including sender, subject, snippet, labels, and message body text, only to decide whether an email should remain in the inbox or be moved to the College Ads label. MailSift may also read Gmail page content in the browser to detect visible message rows and keep the inbox view updated.

MailSift stores user preferences locally in Chrome storage, including protected terms, setup choices, scan status, and a local report of recent filtering activity.

How Data Is Used

MailSift uses Gmail data only to classify and organize messages according to the user's settings. When MailSift identifies a message as college admissions promotional noise, it uses the Gmail API to add the College Ads label and remove the INBOX label. When MailSift identifies a protected message, it keeps that message in the inbox.

Data Sharing

MailSift does not sell user data. MailSift does not transfer user data to third parties. MailSift does not use user data for advertising, creditworthiness, lending, or any purpose unrelated to its inbox filtering function.

Remote Code

MailSift does not use remote JavaScript or remote WebAssembly. The extension code is included in the extension package.

Data Storage

MailSift stores settings and recent scan reports locally using Chrome extension storage. Gmail OAuth access is handled through Chrome Identity and Google's OAuth flow. MailSift does not operate a separate cloud backend.

User Control

Users can edit protected terms, disable automatic scanning, manually scan the inbox, or disconnect Gmail permission from the extension interface. Users can also remove MailSift from Chrome at any time.

Contact

For questions or support, contact the developer through the Chrome Web Store listing.

MailSift is an independent student project and is not affiliated with Gmail, Google, or any college or university.
